A production-ready analytics dashboard service built with FastAPI, TimescaleDB, and Celery. This service provides real-time event tracking, data aggregation, and visualization capabilities.

This service provides high-throughput event ingestion, real-time
analytics, time-series aggregation, and live dashboard updates ---
designed for scalable production environments.
